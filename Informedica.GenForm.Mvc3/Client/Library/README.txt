In order to run the app using ext. A folder named ext should be created in this folder. Next the javascript library should be put in a folder named after the ext version of that ext library.

The current version of the liberary can be found in the index.cshtml files of the different views:
- \Informedica.GenForm.Mvc3\Views\AppTest
- \Informedica.GenForm.Mvc3\Views\Debug
- \Informedica.GenForm.Mvc3\Views\Home
- \Informedica.GenForm.Mvc3\Views\Shared
- \Informedica.GenForm.Mvc3\Views\UnitTest
